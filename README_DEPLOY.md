## Free lifetime hosting option: GitHub Pages

1. Log in to GitHub with username: `Manan0399`.
2. Create a new repository named exactly:
   `Manan0399.github.io`
3. Upload all files and folders from this website folder into the repository root.
4. Go to repository **Settings → Pages**.
5. Under **Build and deployment**, select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Save.
7. Your website should become available at:
   `https://Manan0399.github.io/`
