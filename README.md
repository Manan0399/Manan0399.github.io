## How to publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to repository Settings.
4. Go to Pages.
5. Select:
   - Source: Deploy from branch
   - Branch: main
   - Folder: /root
6. Save.
7. Wait 1-3 minutes.
8. Open your GitHub Pages link.
