## Upload to GitHub

Upload these files directly to the root of your repository:

```text
index.html
style.css
README.md
assets/
resume/
```

Do not upload the parent folder itself.

Then go to:

```text
Settings → Pages → Deploy from branch → main → /root → Save
```
